# Bet Cherry Picker

축구 경기 배당표를 기준으로 **고확률 픽을 선별하고**, 조합 배당률을 목표치까지 끌어올린 뒤, 각 조합의 손익 커버 가능성을 계산하는 MVP 프로젝트입니다.

이 저장소의 목표는 “확정 수익”이나 “무손실 보장”이 아닙니다. 모든 계산은 확률 기반 시뮬레이션이며, 실제 결과는 손실이 발생할 수 있습니다.

## 핵심 기능

- 단일 픽의 시장 암시 확률 계산
- 모델 예상 확률과 배당률 기반 EV 계산
- 고확률·양수 EV 픽 선별
- 2폴더, 3폴더, 4폴더 조합 생성
- 조합 배당률 계산
- 조합 성공확률 계산
- 100원 단위 고정 투입 시 손익 커버 여부 계산
- Dutching 방식의 금액 배분 계산
- S/A/B/C/X 등급 분류

## 빠른 실행

```bash
npm install
npm run dev
```

샘플 결과는 콘솔에 출력됩니다.

## 프로젝트 핵심 전략

```text
고확률 픽 선별
→ 배당률 상승 조합 생성
→ 조합별 성공확률 계산
→ 하나 이상 적중 시 손익 커버 가능성 확인
→ 위험 조합 제외
→ 최종 추천 리스트 출력
```

## 주요 공식

```text
시장 암시 확률 = 1 / 배당률
모델 우위 = 모델 예상 확률 - 시장 암시 확률
EV = 모델 예상 확률 × 배당률 - 1
조합 배당률 = 각 픽 배당률의 곱
조합 성공확률 = 각 픽 예상 확률의 곱 × 상관관계 보정값
적중 반환금 = 투입금 × 조합 배당률
순손익 = 적중 반환금 - 총 투자금
```

## 주의사항

- 본 프로젝트는 분석 도구입니다.
- 수익을 보장하지 않습니다.
- 손실 가능성이 항상 존재합니다.
- 불법 도박 사이트 안내나 우회 방법은 제공하지 않습니다.
- 국내 스포츠토토 이용 시 공식 발매처와 구매 한도, 연령 제한을 준수해야 합니다.
